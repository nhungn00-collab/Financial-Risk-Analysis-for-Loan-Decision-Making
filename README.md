# Financial Risk Analysis for Loan Decision Making

This project analyzes financial data to identify key factors influencing loan default risk and support data-driven credit decisions.

## Overview
- Domain: Financial / Credit Risk
- Goal: Identify high-risk customers and improve loan approval decisions
- Tools: SQL, Python

## Objectives
- Analyze factors affecting loan default risk  
- Identify high-risk customer segments  
- Support better credit decision-making  

## Key Insights
- Customers with lower income and higher debt ratios show significantly higher default risk  
- Loan amount relative to income is a strong predictor of financial risk  
- Certain customer segments consistently exhibit higher risk profiles  
- Risk varies across different financial characteristics  

## Business Impact
The analysis helps financial institutions:
- Identify high-risk applicants  
- Improve loan approval strategies  
- Reduce potential financial losses  
- Support data-driven decision making  

## Tools Used
- SQL (data extraction & transformation)  
- Python (Pandas for analysis)  
- Jupyter Notebook  

## Project Files
- `index.html` – Project website  
- `.ipynb` – Python analysis  
- `.sql` – SQL queries
