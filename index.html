Financial Risk Analysis

Loan Risk Assessment using SQL & Python

View Python Analysis | View SQL Queries

Overview

This project analyzes financial data to identify risk factors affecting loan approval decisions. The goal is to detect high-risk customers and support better credit decision-making.

Objectives

  • Identify key factors influencing financial risk
  • Analyze customer segments with high default probability
  • Support data-driven loan approval decisions

Key Insights

  • Customers with lower income and higher debt show significantly higher risk levels
  • Loan amount relative to income is a strong predictor of default risk
  • Certain customer segments exhibit consistently higher risk profiles
  • Risk distribution varies across different financial characteristics

Business Impact

The analysis helps financial institutions identify high-risk applicants, optimize loan approval strategies, and reduce potential losses by improving risk assessment models.

Tools Used

  • SQL (Data extraction & transformation)
  • Python (Pandas, analysis)
